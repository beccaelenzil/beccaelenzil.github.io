Ada Build - Intro to Programming in Python

  • Project date: 2020 - 2021
  • Technologies: Python, Replit, Colab
  • Ada Build - Intro to Programming in Python

I was the primary developer of the Ada Build - Intro to Python Curriculum. Ada Build is a freely available curriculum intendended to prepare students to apply to Ada Developers Academy.

Ada Developers Academy is a cost-free coding school for women and gender expansive adults. Ada primarily serves Black, Latine, Indigenous Americans, Native Hawaiian & Pacific Islander, LGBTQIA+, and low-income people.

For the Ada Build Curriculum, I chose the colab platform, wrote text book content, developed projects and solutions, developed study guides for live review sessions, and recorded livecode video solutions.
